⚙️ Google Drive 연결 설정

📋 사용 방법

  1. Google Cloud Console에 접속
  2. 프로젝트 생성 후 Google Drive API 활성화
  3. 사용자 인증 정보 → API 키 만들기
  4. API 키 제한사항 설정:
    • 애플리케이션 제한: HTTP 리퍼러
    • 웹사이트 제한: https://[your-github-username].github.io/*
  5. Google Drive 폴더를 "링크가 있는 모든 사용자"로 공유
